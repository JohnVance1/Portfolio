Procedural Cave Generation

November 2021 - Current

Working on a procedural cave generator using a marching squares algorithm. The current build includes random, perlin, and fractal brownian noise generation methods and a top to bottom A star pathfinding algorithm. Currently in development is better pathfinding algorithms and an option for 2D platformer style pathfinding as well as item and enemy genereations.
